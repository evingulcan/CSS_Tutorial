Görüntü Şeffaflığı(Transparency)

opacity özelliği, bir elemanın şeffaflığını belirtir. Değer ne kadar düşükse, o kadar şeffaftır:

%50 opaklığa sahip resim:

Transparent Box

With opacity:

10% opacity

30% opacity

60% opacity

opacity 1

With RGBA color values:

10% opacity

30% opacity

60% opacity

default

Opaklık özelliğini kullanırken metnin nasıl saydamlaştığına ve arka plan rengine dikkat edin.

Bu, şeffaf kutuya yerleştirilmiş bazı metinlerdir.
